295  /* /\** This method simply calls the appropriate method in */
296  /* PSMParticleEntropyFunction. It is exposed here for */
297  /* convenience. The minimum radius of the neighborhood of points */
298  /* that are considered in the entropy calculation. The neighborhood */
299  /* is a spherical radius in 3D space. The actual radius used in a */
300  /* calculation may exceed this value, but will not exceed the */
301  /* MaximumNeighborhoodRadius. This parameter should ALWAYS be set by */
302  /* an application, because it must be scaled according to the */
303  /* spacing in the image. A good value is typically 5x the spacing */
304  /* of the highest-resolution dimension (the dimension with the */
305  /* smallest spacing. *\/ */
306  /* void SetMinimumNeighborhoodRadius( double s) */
307  /* { this->m_ParticleEntropyFunction->SetMinimumNeighborhoodRadius(s); } */
308  /* double GetMinimumNeighborhoodRadius() const */
309  /* { return this->m_ParticleEntropyFunction->GetMinimumNeighborhoodRadius(); } */
310 
311  /* /\** Maximum radius of the neighborhood of points that are considered */
312  /* in the calculation. The neighborhood is a spherical radius in 3D */
313  /* space. MaximumNeighborhoodRadius should be set to a value */
314  /* equal-to or less-than the length of the largest dimension of the */
315  /* image. This parameter should ALWAYS be set by an application, */
316  /* since this class cannot know by default the size of input */
